# Code of Conduct

## Expected Behavior

Contributors are expected to:

- communicate directly and respectfully
- focus discussion on the code, behavior, and user impact
- accept review feedback without turning it personal
- avoid harassment, discrimination, or abusive conduct

## Unacceptable Behavior

The following are not acceptable in project spaces:

- personal attacks
- threats or intimidation
- harassment or stalking
- discriminatory language or conduct
- publishing someone else's private information without permission
- knowingly submitting malware, credential leaks, or harmful payloads

## Enforcement

Project maintainers may remove comments, close discussions, reject contributions, or block participation when behavior violates this policy.

## Reporting

Report conduct issues privately to the repository owner instead of using public issues when privacy or safety is involved.
